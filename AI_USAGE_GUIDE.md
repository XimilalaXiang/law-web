# 🤖 AI防骗助手使用指南

## 📱 用户界面

### 1. 悬浮按钮

在所有页面的右下角，你会看到一个蓝色的圆形悬浮按钮：

- 💬 点击即可打开AI助手
- 悬浮按钮采用Neumorphic设计，与网站风格统一
- 如果有新消息会显示红色提示点

### 2. 聊天窗口

点击悬浮按钮后，会弹出聊天窗口（420x600px）：

**头部功能**：
- 🤖 AI防骗助手标识
- 🗑️ 清空历史按钮
- ❌ 关闭窗口按钮

**消息区域**：
- 用户消息：蓝色气泡，右侧显示
- AI回复：白色气泡，左侧显示，支持Markdown格式
- 实时打字效果，更自然的交互体验

**输入区域**：
- 多行文本输入框
- Enter 发送，Shift+Enter 换行
- 📤 发送按钮

## 💡 使用场景

### 场景1：分析招聘信息

**步骤**：
1. 打开AI助手
2. 点击"帮我分析招聘信息"或直接输入
3. 粘贴招聘信息内容
4. AI会分析风险等级并给出建议

**示例输入**：
```
我看到一个招聘信息：
岗位：客服专员
薪资：8000-12000元/月
要求：先参加公司培训，需缴纳3000元培训费
承诺：培训后保证录用

这个靠谱吗？
```

**AI会给出**：
- 🔴 风险等级：高风险
- ⚠️ 可疑点分析
- 💡 防范建议
- 📚 相似案例引用

---

### 场景2：了解诈骗类型

**快速提问**：
- "什么是培训贷？"
- "如何识别黑中介？"

**AI回复示例**：
```markdown

**定义**：

**典型特征**：
- ⚠️ 承诺"边学边赚""先学后付"
- ⚠️ 诱导办理分期贷款
- ⚠️ 课程质量低劣
- ⚠️ 无法提供承诺的就业机会

**真实案例**：

**防范建议**：
1. 警惕要求贷款支付的培训
2. 通过官方渠道核实培训机构资质
3. 签订合同前仔细阅读条款
```

## 🔧 高级技巧

### 1. 详细提问获得更好答案


## ⚡ 常见问题

### Q1: AI回复速度慢？

**原因**：
- 网络延迟
- API服务繁忙
- 问题复杂度高

**建议**：
- 简化问题描述
- 等待流式响应完成
- 避免重复发送

### Q2: AI回复不准确？

**原因**：
- 问题描述不够清晰
- 缺少关键信息

**建议**：
- 提供更多细节
- 使用快速提问模板
- 明确说明你的困惑点

### Q3: 对话历史丢失？

**原因**：
- 清除了浏览器缓存
- 使用了隐私/无痕模式

**解决**：
- 使用正常模式浏览
- 重要对话及时截图保存

### Q4: 发送失败怎么办？

**检查**：
1. 网络连接是否正常
2. 输入内容是否为空
3. 是否正在等待上一条回复

**重试**：
点击错误提示中的"重试"按钮
